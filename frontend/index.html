About this app

This is the Spaced Repetition Topic Recommender described in Will Daly's final-project papers for AAI 5015 (Mathematical Concepts, Northeastern). It implements all three mathematical layers end-to-end:

  1. Part 1 — Bayesian updating. For each topic, P(retains | quiz evidence) is updated after every quiz attempt, with an exponential time-decay prior controlled by a per-topic decay rate α.
  2. Part 2 — Linear regression. The model ŷ = β₀ + β₁ x is fit via closed-form OLS on your quiz history (x = days since review, y = score).
  3. Part 3 — Gradient descent. Batch GD and SGD fit the same model iteratively; you can vary the learning rate and watch the loss curve. This is what a self-tuning version of the assistant would use to learn α per student.

Data

Study content comes from the AAI 5015 Canvas course — modules 6–10 (Probability 1, Probability 2, Statistics 1–3). Quiz questions are generated by Claude Haiku 4.5 from the extracted topic content during scripts/seed.py.
